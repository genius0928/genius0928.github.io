# 천각 블로그
천재개발자들의 모각코 모임, 천각의 블로그입니다. 

## requirement 
 - https://rubyinstaller.org/downloads/ (윈도우)
 - brew install ruby (맥)
 - gem install jekyll bundler
 - bundle install 

## 테스팅
```
bundle exec jekyll serve
```
## 빌드 
```
bundle exec jekyll build
```  

표랑 그래프
리스트 링크 버그 수정
